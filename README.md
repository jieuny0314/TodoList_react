# 사용 스택(라이브러리 등)
- React

    ㄴ react-spinners(로딩 화면)
    
    ㄴ react-icons
    
    ㄴ react-router-dom
- Redux (메모 목록 관리)
- Json-server (할일 목록 관리)
- Styled-components
- Framer-motion (페이지 이동 효과)
- Moment (달력)

# 구현 기능
- todo CRUD
- todo 초기화 
- 달력에서 완료 todo 확인 기능

    ㄴ 완료한 일정이 있는 날은 빨간 점으로 표시
- memo CRUD
- 회원가입, 로그인, 로그아웃 기능

    ㄴ json-server 이용
- 글꼴 변화 기능

    ㄴ 아직 전역으로 설정은 안된다..
- 데이터 받아오기 전까지 로딩화면 구현
- Menu 기능 구현
